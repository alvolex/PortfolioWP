This is a solo project that mimics a lot of what MineCraft does, it’s a voxel-world that is generated using procedural generation, namely using several Fractional Brownian Motions (FBM’s) for generation and 3D FBM’s to generate cave systems.<\/p>\n

To get good frame rates in the game the generator is not using any primitives for generating the world, instead it’s building the blocks out of quads. This way we can also combine all the blocks in a chunk to be a single mesh and any quad that isn’t visible will not be rendered.<\/p>\n\n\t\t
